Backup Contents

Included in every backup:
  • carecall.db — all clients, schedules & call history
  • carecall_jobs.db — scheduled job state
  • .env — Twilio credentials & configuration
  • carecall.service — systemd service definition
  • uploads/ — all recorded audio files
  • backup_config.json — backup schedule settings

Application code is not included — it can be restored from GitHub.
